# HomeschoolHub

This is the official website for **HomeschoolHub**, focused on supporting homeschooling families in Kenya.

The site introduces HomeschoolHub's services, including:
- personalized, CBC-aligned learning support
- guidance from Kenyan educators
- flexible homeschooling pathways for families
- a supportive parent learning community

It also provides a contact channel where families can request a free consultation.
